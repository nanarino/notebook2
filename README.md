## 本地運行

安裝開發環境

```bash
pnpm i
```

啟動腳手架

```bash
# serve with hot reload at localhost:5173
pnpm docs:dev

# build for production with minification
pnpm docs:build
```

本地運行可以將配置裡的 base 改為 `'/'`

```diff
- const base = "/notebook2/"
+ const base = "/"
```
